### 满大街的重写CNode，这个有什么亮点？

- 比较接近原生APP体验
- 流畅的加载过渡效果
- 舒服细腻的UI样式布局
- 合理的列表渲染优化

### 使用方法
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
